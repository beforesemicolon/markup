Installation

The templating system is a plug-and-play package which means you can either use the CDN or npm to install it. There is no need to any additional setup or requirements.

Via CDN

This method is the quickest loading option and can be placed in the head tag of the document. 

<script src="https://unpkg.com/@beforesemicolon/markup/dist/client.js" />

You may also specify a specific version you want.

 <script src="https://unpkg.com/@beforesemicolon/markup@1.0.0/dist/client.js"/>

You can use various CDN providers like unpkg, jsDelivr, and more.

Accessing content

The client CDN link will create a global variable you can access for all the internal functions. 

const {html} = BFS;

Via npm

This package is also available via npm which will allow you to use it in server JavaScript environments 

npm install @beforesemicolon/markup

Accessing content

import {html} from "@beforesemicolon/markup";

Via yarn

yarn add @beforesemicolon/markup

Typescript

This package was built using typescript. There is no need to install a separate type package for it. All types are exported with it.
